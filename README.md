# MyVScodeProfile
VSCode Setup
Install Extensions
cat extensions.txt | xargs -L 1 code --install-extension
Apply Settings
•	Copy settings.json to:
o	Windows: %APPDATA%\Code\User
o	macOS: ~/Library/Application Support/Code/User
o	Linux: ~/.config/Code/User
•	Copy keybindings.json to same folder
Optional
Restart VS Code after setup
